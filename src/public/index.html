Link Shortener

Open source link shortener built using Node

How to

? = Optional

Open shorten link

GET /:code

RESPONSE 302 URL

RESPONSE 404 application/json {
message: String
}

RESPONSE 422 application/json {
message: "Invalid code"
}

Create a new code

POST /

BODY application/json {
url: String<URL>,
?code: String,
?expire: Int
}

RESPONSE 200 application/json {
code: String,
accessCode: String<Hash>
}

RESPONSE 409 application/json {
message: "This code is already used"
code: String,
}

RESPONSE 422 application/json {
errors: Object<Array>
}

Delete existing code

DELETE /

BODY application/json {
code: String,
accessCode: String<Hash>,
}

RESPONSE 200 application/json {
message: "Successfully removed the URL from database"
}

RESPONSE 403 application/json {
message: "Missing access to delete this code"
}

RESPONSE 404 application/json {
message: "No URL with this code registered"
}
